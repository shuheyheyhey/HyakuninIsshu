# ミスター百人一首

「ミスター百人一首」は、百人一首100首の読み上げ練習をサポートする iOS アプリです。

## 機能

### 閲覧・管理
- 上の句・下の句に分かれたカード一覧表示
- カードへの音声録音と再生（未録音時は Apple TTS で自動読み上げ）
- タグによるカードのグルーピングと色分け管理

### プレイ
- 全カードまたはタググループ単位でランダム読み上げ
- 上の句読み上げ → インターバル → 下の句読み上げ を全カードで繰り返し
- 読み上げ途中のスキップ
- インターバル秒数の設定

## 技術スタック

| 項目 | 採用技術 |
|------|---------|
| UI | SwiftUI |
| データ永続化 | SwiftData（ユーザーデータのみ） |
| 歌テキスト | `Poems.strings`（named strings table） |
| 音声再生 | AVAudioPlayer / AVSpeechSynthesizer |
| 音声録音 | AVAudioRecorder |
| 非同期処理 | Swift Concurrency（async/await、Actor） |

## 動作環境

- iOS 18.0+
- Xcode 26.6+
- Swift 6.3+

## データ設計

### 保存先の分担

**SwiftData**（ユーザーデータ）
- `Card`: カードID・句ID・所属タググループ
- `TagGroup`: グループ名・色（#RRGGBB）・カードとの多対多リレーション

**Poems.strings**（マスタデータ、読み取り専用）
- キー形式: `poem_{n}_upper.text`, `poem_{n}_upper.reading`, `poem_{n}_lower.text`, `poem_{n}_lower.reading`, `poem_{n}_author`

**ファイルシステム**（録音音声）
- 保存先: `Documents/Audio/{phraseID}.m4a`（決め打ちパス）
- ファイル有無で録音登録判定、最大15秒

## セットアップ

Xcode でプロジェクトを開きビルドするだけで動作します。初回起動時に100首のカードが自動生成されます。

## 権限

- **マイクロフォン**: 各カードへの音声録音に使用（録音機能を使わない場合は不要）

## 開発方針

- コミット粒度は機能単位で小さく
- ロジック層（モデル・音声・プレイ進行）には単体テストを整備
- ビューにロジックを書かず、`@Observable` モデルに分離
